# D-veloppement-de-la-plateforme-SchoolPrepar-avec-Symfony

Plateforme d’orientation scolaire développée avec Symfony.

## Installation

1. Cloner le projet
2. Installer les dépendances :
   composer install
3. Lancer le serveur :
   symfony server:start

## Architecture

- MVC respecté
- Controllers séparés
- Templates Twig organisés
- Git versionné
